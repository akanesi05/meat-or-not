# meat-or-not
Djangoで作る食材判定クイズアプリ
# 渡辺さんの嫌いな肉クイズ (niku-kirai-quiz)

職場で大変お世話になっている渡辺さんは、肉が嫌いです。  
特に「鳥」が苦手で、帰宅時に飛んでいる鳥を見ると怯え、  
モニターに鳥が映ると今日も叫んでいました。  

でも、不思議なことに「食べられる肉」もあるそうです。  
肉好きの私としては思想が真逆で、そのこだわりが面白いと思い、  
クイズアプリとして実装してみました。  

シンプルなクイズ形式ですが、遊びながら語彙力アップにも（？）役立ちます。

---

## Features
- 出題された食べ物が「渡辺さんにとって肉かどうか」を ⭕️❌ で答えるクイズ
- 渡辺さんが嫌いな肉（特に鳥）が含まれているかを判定
- Django を使用したバックエンド
- （将来的に React でフロントを実装予定）
