# 渡辺さんの肉か肉じゃないか 判定アプリ

このプロジェクトは、ユーザーが食べ物を入力し、それが肉か肉でないかを判定するアプリケーションです。最終的にはクイズ形式の機能も追加予定です。

## 構成

- `watanabe_meat_judge/`: プロジェクトのメインディレクトリ
  - `watanabe_meat_judge/`: Djangoプロジェクトの設定ファイルが含まれています。
  - `judge/`: 判定アプリケーションのロジックが含まれています。
  - `manage.py`: Djangoプロジェクトを管理するためのコマンドラインツールです。

## セットアップ手順

1. **仮想環境の作成**  
   Pythonの仮想環境を作成します。
   ```
   python -m venv venv
   ```

2. **仮想環境のアクティベート**  
   Windows:
   ```
   venv\Scripts\activate
   ```
   macOS/Linux:
   ```
   source venv/bin/activate
   ```

3. **依存関係のインストール**  
   Djangoをインストールします。
   ```
   pip install django
   ```

4. **データベースの設定**  
   `watanabe_meat_judge/settings.py`でデータベースの設定を行います。

5. **マイグレーションの実行**  
   データベースのマイグレーションを実行します。
   ```
   python manage.py migrate
   ```

6. **サーバーの起動**  
   開発用サーバーを起動します。
   ```
   python manage.py runserver
   ```

## 使用方法

1. ブラウザで `http://127.0.0.1:8000/` にアクセスします。
2. 食べ物の名前を入力し、「判定」ボタンをクリックします。
3. 判定結果が表示されます。
